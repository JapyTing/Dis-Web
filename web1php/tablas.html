Las tablas

Con los TAG`S TABLE, TR, TH y TD se crean tablas que organizan la informaciòn para resumirla.

  • TR: Permite declarar renglones (rows)
  • TH: Es para declarar cabezeras de columnas
  • TD: Para declarar los datos

Los atributos: border,caption, colspan y rowspan

  • BORDER: Establece el ancho del borde para la tabla
  • CAPTION: Establece un titulo arriba de la cebecera
  • COLSPAN: Permite que una celda ocupe varias columnas
  • ROWSPAN: Permite que una celda ocupe varios renglones
